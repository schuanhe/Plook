# 项目介绍

这是一个基于Vue.js的视频播放器，支持多人在线观看同一视频，并且可以进行实时聊天。该项目使用了WebSocket实现了多人在线聊天和视频播放的同步，使用了Element-UI作为UI框架，使用了video.js作为视频播放器。

## 项目功能

- 视频播放：支持多人在线观看同一视频，支持视频的播放、暂停、快进、快退、切换视频源等功能。
- 实时聊天：支持多人在线聊天，支持发送文字消息。
